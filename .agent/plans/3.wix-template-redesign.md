# 3. Landing Redesign — "Urban Nomad Adventures" (Wix template 2297) look

✅ **Simple** — styling/layout pass only, no backend or route changes.

## Reference

Wix "Adventure Tour Company" template (demo: wix.com/demone2/tours-company), inspected
live via browser. Key traits: editorial minimal design, Playfair-style serif display
headings, black/white + warm beige (#EFE9E3) palette, full-bleed photography, thin-line
icons, square outlined buttons, generous whitespace.

## Tasks

1. Fonts + theme: add Playfair Display via next/font as `--font-heading`; shift light
   theme secondary/muted tones to warm beige; default color scheme to light (template
   is light-first), dark toggle still works.
   - **Validation**: headings render in serif; beige sections visible; toggle still flips.
2. Header: line-art triangle logo mark + serif brand left; small nav right, matching
   template weight/spacing. Applies to all pages via SiteHeader.
   - **Validation**: header matches on /, /plan, /dashboard, /login.
3. Landing page rebuild, in template section order:
   hero (full-viewport Unsplash photo, dark overlay, huge white serif title, subtitle,
   outlined "Let's Go" button scrolling to planner) → beige story section → white
   3-feature section w/ thin-line lucide icons → planner section (the trip search form,
   restyled square/minimal, black submit) → 3-photo gallery band → beige testimonials
   (3 serif quotes) → photo quote band → thin beige footer.
   - **Validation**: browser screenshots compare section-by-section against template;
     search form still navigates to /plan and generates an itinerary.
4. next.config: allow images.unsplash.com for next/image.
   - **Validation**: hero/gallery images load with no console errors.
